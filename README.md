# ETS2 Config Manager

Репозиторий для управления модами и конфигурациями Euro Truck Simulator 2.

## Содержимое
- Моды физики, грузов и совместимости
- Скрипт проверки `checker.py`
- Инструкция по установке

## Ветки
- `main` — стабильная версия модов
- `praktika_4_ets2` — тестовая ветка для практической работы №4
